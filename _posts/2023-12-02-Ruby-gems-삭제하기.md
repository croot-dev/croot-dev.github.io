```shell
gem uninstall -aIx
```


> **Option Tip!**  
> `a` - removes all versions  
> `I` - ignores dependencies  
> `x` - includes executables


```shell
for i in `gem list --no-versions`; do gem uninstall -aIx $i; done
```
